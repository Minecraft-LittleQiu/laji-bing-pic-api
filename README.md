# 垃圾 必应 日图 API（每日图片）
## 这个 API 干什么用？
获得 Bing 今日的图片。版权来自 Bing
## 与原版有什么区别呢？
更加精简。
## 如何使用？
本 API 指向一个图片。运行要求是：PHP 5.2.0 以上，支持 JSON 的
## 这个 API 和其他的有什么不一样吗？
•代码仅有 15 行。
•精，小，简，便。
•配合 Wallpaper Engine 食用绝佳。
•使用 302 状态码做跳转，使用 HTTPS
•下载一次，到处运行，永远享受。
程序原作者为 [@Little_Qiu](https://www.littleqiu.net)。 同样地，使用 GPL3.0 开源方法。
